# 智慧图书管理系统 java356

## 项目概述

智慧图书管理系统是一款基于Springboot和Vue的前后端分离的Java项目。系统致力于提供高效的图书管理功能，通过精细化的角色划分，实现图书的高效借阅与归还流程。

## 技术栈

- **后端技术**：Springboot、Mybatis-Plus
- **前端技术**：HTML、CSS、Layui、jQuery、Bootstrap、Vue、axios、Element-UI
- **开发工具**：IDEA 2020.3以上、VSCode
- **运行环境**：
  - JDK 1.8
  - Maven 3.6.0
  - MySQL 5.7
  - NodeJS 14

## 功能模块

### 系统角色

- **管理员**
  - 图书管理
  - 图书留言管理
  - 图书借阅记录管理
  - 归还图书
  - 论坛管理
  - 读者管理
  - 轮播图管理
  - 基础数据管理
  - 书架管理
  - 图书类型管理
  - 个人中心管理
    - 个人信息管理
    - 密码修改
  - 读者类型管理

- **读者**
  - 图书借阅
  - 图书留言
  - 图书收藏
  - 查看论坛帖子
  - 发布帖子
  - 查看帖子
  - 评论帖子
  - 个人中心管理
    - 个人信息更改
    - 图书收藏管理
    - 图书借阅管理
    - 查询功能
